# Simple CRUD

**Projeto**

O projeto é um CRUD para gerenciamento de uma tabela no banco. Tendo o objetivo de me ajudar a compreender na pratica como que funciona a manipulação do banco com os dados inseridos.


**CRUD**

Seguindo fielmente o conceito de CRUD podemos adicionar , listar , atualizar e deletar cada usuario do banco de dados.

**Tecnologias**

- [NodeJS](https://nodejs.org)
- [Nodemon](https://nodemon.io/)
- [Express](https://expressjs.com/pt-br/)
- [Knex](http://knexjs.org/)
- [MySQL](https://www.mysql.com/)
- [Insominia](https://insomnia.rest/)
- [Xammp](https://https://www.apachefriends.org/pt_br/index.html)

**Como usar**

- Tenha instalado na maquina Node, Insomnia e Xammp. 
- Faça clone do projeto e entre na pasta
- Digite `npm install -i`
- Inicie o servidor Xammp
- Ditite no terminal o seguinte comeando : `npx knex migrate:latest`
- Para rodar o projeto utilize `npm dev`
- Configure as rotas do Insominia seguindo get , post , put e delete com os nomes respectivos : read, create, update e delete
